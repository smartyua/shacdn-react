# Skill: documentation

## When to use
Updating human or AI docs

## Inputs
Topic; source of truth files

## Steps
1. Prefer linking to code over duplicating it
2. Update docs/ai or .ai/knowledge as appropriate
3. Mark unknown sections Status: Not documented yet
4. Never invent APIs

## Required checks
- paths exist

## Expected result
Accurate docs

## Forbidden
- Placeholder fiction

## Done when
Docs match code
